# Product Context

## Why does this project exist?
The current Trade Finance systems require a flexible yet strictly compliant solution for banking regulations. The Import LC project aims to provide a centralized management tool to reduce manual errors and accelerate transaction processing.

## Target Users
1.  **Maker (Branch/CSR):** Inputs LC data, amendment requests, or registers document presentations.
2.  **Checker (Operations/IPC):** Approves requests from the Maker, ensuring accuracy and compliance.
3.  **Admin:** Manages system configuration, categories, and user accounts.

## Desired User Experience
- Intuitive interface divided into functional tabs (MainLC, Amendments, Drawings, Financials).
- Real-time alerts for mandatory fields or risks (limits, provisions).
- Transparent approval workflows with detailed history (`LcHistory`).
