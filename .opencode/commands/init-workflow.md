# Init Workflow Command

## Description
Initializes workflow directory structure for TradeFinance development.

## Usage
```
/init-workflow
```

## Actions
1. Creates `.opencode/workflow/` directory if not exists
2. Creates `README.md`
3. Creates `state.md`
4. Creates `QUICK_REFERENCE.md`

## Files Created
- `.opencode/workflow/README.md`
- `.opencode/workflow/state.md`
- `.opencode/workflow/QUICK_REFERENCE.md`

## Example
```
> /init-workflow
✓ Workflow directory initialized
✓ README.md created
✓ state.md created
✓ QUICK_REFERENCE.md created
```
