# SWIFT MT700 - Message Type 700: Issue of a Documentary Credit

## 1. Overview
This document outlines the detailed field specifications for the SWIFT MT700 message, used for issuing a documentary credit.

## 2. General Validation Rules
- **SWIFT Character Set X**: All alphanumeric fields MUST only allow characters: `A-Z`, `a-z`, `0-9`, `/`, `-`, `?`, `:`, `(`, `)`, `.`, `,`, `'`, `+`, and `space`.
- **Date Format**: Standard ISO dates (YYYY-MM-DD), validated as per UCP 600 (e.g., Expiry > Issue).
- **Mandatory Fields**: Fields marked as mandatory must be filled to proceed with LC issuance.
